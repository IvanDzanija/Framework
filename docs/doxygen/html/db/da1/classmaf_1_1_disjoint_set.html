Detailed Description

template<typename T>
class maf::DisjointSet< T >

Implements a Disjoint Set Union (DSU) data structure.

This class tracks a collection of elements partitioned into a number of disjoint (non-overlapping) sets. It is also known as a Union-Find data structure.

This implementation acts as a container, "owning" the elements added to it. It provides efficient operations to add new sets, merge existing sets, and determine if two elements are in the same set.

Operations are performed using lightweight size_t handles, which are returned when elements are added. This avoids copying or hashing the elements T for DSU operations, making this implementation very efficient for large or complex element types.

It uses two optimizations for amortized O(α(n)) time complexity:

  1. Union by Size: Merges the smaller set into the larger one.
  2. Path Compression: Flattens the tree structure during find operations.

More information: https://cp-algorithms.com/data_structures/disjoint_set_union.html
