Detailed Description

template<typename T>
class maf::math::Matrix< T >

A general-purpose, row-major, dense matrix class.

This class implements a matrix with contiguous row-major storage. It supports a wide range of arithmetic operations, constructors, and utility methods.

The implementation is templated to support various numeric types (T). Many operations (like multiplication and addition) are parallelized using OpenMP and employ blocking strategies for cache efficiency.

Template Parameters
TThe numeric type of the matrix elements (e.g., float, double, int).
