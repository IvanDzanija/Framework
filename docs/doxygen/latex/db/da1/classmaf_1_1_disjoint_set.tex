\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class maf\+::\+Disjoint\+Set$<$ T $>$}
Implements a Disjoint Set Union (DSU) data structure. 

This class tracks a collection of elements partitioned into a number of disjoint (non-\/overlapping) sets. It is also known as a Union-\/\+Find data structure.

This implementation acts as a container, "{}owning"{} the elements added to it. It provides efficient operations to add new sets, merge existing sets, and determine if two elements are in the same set.

Operations are performed using lightweight {\ttfamily size\+\_\+t} handles, which are returned when elements are added. This avoids copying or hashing the elements {\ttfamily T} for DSU operations, making this implementation very efficient for large or complex element types.

It uses two optimizations for amortized O(Î±(n)) time complexity\+:
\begin{DoxyEnumerate}
\item Union by Size\+: Merges the smaller set into the larger one.
\item Path Compression\+: Flattens the tree structure during {\ttfamily \doxylink{classmaf_1_1_disjoint_set_a8ba4ca7bf154318426dbafbbe294c6ba}{find}} operations.
\end{DoxyEnumerate}
